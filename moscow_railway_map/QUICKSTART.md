# Инструкция по быстрому старту

## Быстрый запуск в VS Code

### 1. Откройте проект
Откройте папку `moscow_railway_map` в VS Code.

### 2. Установите расширения (рекомендуется)
VS Code предложит установить рекомендуемые расширения:
- C/C++ (ms-vscode.cpptools)
- CMake Tools (ms-vscode.cmake-tools)

Нажмите "Install All" для установки.

### 3. Настройте сборку
1. Откройте палитру команд: `Ctrl+Shift+P` (Linux/Windows) или `Cmd+Shift+P` (macOS)
2. Введите "CMake: Configure" и нажмите Enter
3. Выберите компилятор (GCC или Clang)

### 4. Соберите проект
1. Снова откройте палитру команд
2. Введите "CMake: Build" и нажмите Enter
3. Или нажмите `Ctrl+Shift+B` для сборки по умолчанию

### 5. Запустите приложение
1. Откройте палитру команд
2. Введите "CMake: Debug" или "CMake: Run Without Debugging"
3. Или найдите исполняемый файл в папке `build/` и запустите его

## Ручная сборка из терминала

```bash
cd /workspace/moscow_railway_map

# Создание папки сборки
mkdir build
cd build

# Конфигурация
cmake ..

# Сборка
make -j$(nproc)

# Запуск
./MoscowRailwayMap
```

## Проверка зависимостей

### Qt 6
Проверьте установлен ли Qt 6:
```bash
qmake6 --version
# или
pkg-config --modversion Qt6Core
```

Если Qt 6 не установлен:

**Ubuntu/Debian:**
```bash
sudo apt-get install qt6-base-dev libqt6widgets6
```

**Fedora:**
```bash
sudo dnf install qt6-qtbase-devel
```

**Arch Linux:**
```bash
sudo pacman -S qt6-base
```

### CMake
Проверьте версию CMake:
```bash
cmake --version
```

Требуется версия 3.16 или выше.

**Ubuntu/Debian:**
```bash
sudo apt-get install cmake
```

## Возможные проблемы и решения

### Ошибка: "Qt6 not found"
Убедитесь, что Qt 6 установлен и CMake может его найти:
```bash
cmake -DCMAKE_PREFIX_PATH=/path/to/qt ..
```

### Ошибка компиляции
Убедитесь, что у вас установлен компилятор с поддержкой C++17:
```bash
g++ --version
```

### Ошибка запуска приложения
Убедитесь, что библиотеки Qt доступны в PATH:
```bash
export LD_LIBRARY_PATH=/path/to/qt/lib:$LD_LIBRARY_PATH
```

## Основные возможности

1. **Просмотр карты** - перемещение мышью
2. **Масштабирование** - колесо мыши или кнопки +/- 
3. **Выбор станции** - клик по маркеру
4. **Смена режима** - выпадающий список справа
5. **Информация о станции** - отображается в правой панели

## Добавление данных

Для добавления новых станций отредактируйте файл `src/railwaydata.cpp` и добавьте данные в функцию `loadMoscowRingDistrictData()`.

Пример:
```cpp
m_stations.append({
    "Новая станция",
    QPointF(55.75, 37.65),  // Координаты
    "passenger",             // Тип
    "4 пути",                // Количество путей
    "Постоянный ток",        // Электрификация
    "Описание для монтера"   // Примечание
});
```
