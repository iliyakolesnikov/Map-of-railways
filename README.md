# Интерактивная карта железных дорог Москвы

## Описание проекта
Проект представляет собой интерактивную карту железнодорожной инфраструктуры Москвы с возможностью переключения режимов отображения и масштабирования.

## Структура проекта

```
moscow_railway_map/
├── CMakeLists.txt          # Файл сборки CMake
├── README.md               # Документация проекта
├── include/                # Заголовочные файлы
│   ├── mainwindow.h        # Главное окно приложения
│   ├── mapwidget.h         # Виджет карты
│   ├── railwaydata.h       # Модель данных железных дорог
│   └── stationmarker.h     # Маркер станции на карте
├── src/                    # Исходный код
│   ├── main.cpp            # Точка входа в приложение
│   ├── mainwindow.cpp      # Реализация главного окна
│   ├── mapwidget.cpp       # Реализация виджета карты
│   ├── railwaydata.cpp     # Данные о железных дорогах
│   └── stationmarker.cpp   # Реализация маркера станции
├── resources/              # Ресурсы (иконки, изображения)
├── data/                   # Дополнительные данные (JSON, XML)
└── ui/                     # UI файлы (если используются .ui файлы)
```

## Функциональность

### Реализованные возможности:
1. **Отображение железных дорог** - показаны основные направления и МКЖД
2. **Станции разных типов**:
   - Технические (красные маркеры)
   - Узловые (синие маркеры)
   - Пассажирские (зелёные маркеры)
3. **Масштабирование карты**:
   - Колесом мыши
   - Кнопками "+" и "-"
   - Кнопкой "Сброс"
4. **Информация о станции** - при клике показывается:
   - Название
   - Тип станции
   - Количество путей
   - Электрификация
   - Примечания для монтера путей
5. **Переключение режимов** - кнопка выбора режима отображения
6. **Адаптивное отображение** - маркеры скрываются/появляются при изменении масштаба

### Режимы отображения:
- **Железные дороги** (основной режим) - показывает пути и станции
- **Высота зданий** (заготовка для будущего расширения)
- **Транспорт** (заготовка для будущего расширения)

## Сборка и запуск

### Требования:
- Qt 6.x
- CMake 3.16+
- C++ компилятор с поддержкой C++17

### Инструкция по сборке:

```bash
# Создание директории сборки
mkdir build
cd build

# Конфигурация проекта
cmake ..

# Сборка
cmake --build .

# Запуск
./MoscowRailwayMap
```

### В VS Code:
1. Откройте папку проекта в VS Code
2. Установите расширение "CMake Tools"
3. Нажмите "Configure" в нижней панели
4. Нажмите "Build" для сборки
5. Нажмите "Run" для запуска

## Расширение функциональности

### Добавление новых станций:
Откройте `src/railwaydata.cpp` и добавьте новые записи в функцию `loadMoscowRingDistrictData()`:

```cpp
m_stations.append({
    "Название станции",
    QPointF(широта, долгота),
    "тип (technical/junction/passenger)",
    "кол-во путей",
    "электрификация",
    "примечание"
});
```

### Добавление новых режимов:
1. Добавьте новый режим в `mainwindow.cpp` в `setupUI()`:
   ```cpp
   m_modeComboBox->addItem("Новый режим", "newmode");
   ```

2. Обновите обработку в `mapwidget.cpp` в `updateMarkersVisibility()`

### Добавление данных о зданиях:
Поместите данные в папку `data/` в формате JSON или XML и загрузите их в соответствующем режиме.

## Техническая информация

### Используемые технологии:
- **Qt Widgets** - графический интерфейс
- **QGraphicsView/QGraphicsScene** - отображение карты
- **CMake** - система сборки
- **C++17** - стандарт языка

### Архитектура:
- **MVC паттерн** - разделение данных (RailwayData), представления (MapWidget) и контроллера (MainWindow)
- **Событийно-ориентированная модель** - использование сигналов и слотов Qt
